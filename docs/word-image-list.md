```

シンプルな背景（白一色）、太めの輪郭線、
パステルカラー、子ども向け、正面向き、影は最小限、
被写体1つのみを画面中央に大きく、テキストや文字は入れない。
被写体: {ここに下表の「被写体」を入れる}
```

## ファイル名・保存先
- ひらがな → `assets/images/words/hiragana/{ファイル名}.png`
- カタカナ → `assets/images/words/katakana/{ファイル名}.png`
- 正方形、512×512か1024×1024、背景白 または 透過PNG

## 備考
- ✅済＝既に生成済み。**ただしファイル名を修正しました**：以前は単語名（`shika.png`等）でしたが、統一のため「かなのローマ字」（`shi.png`等）に変更しています。もし既に`shika.png`等の名前で保存済みなら、`shi.png`のように**リネームして** `assets/images/words/hiragana/` に置いてください（絵自体はそのまま使えます）。
- コ／レ／ワ／ヲ／ンの5つは、ひらがな版と同じ絵を使い回してOKとしています（無理に別の絵を用意するとかえって不自然になるため）。省力化にもなります。
- を・ん・ヲ・ンは前述の理由で「〜を使う／〜で終わる」という例外的な扱いです。プロンプトの「被写体」欄はそのまま使えますが、被写体名の説明が少し変則的なので、そこだけ意識してください。
- 92枚は多いので、優先順位を付けるなら「あかさたなはまやらわ」の各行の最初の文字（あ・か・さ・た・な・は・ま・や・ら・わ）から始めると、進捗が体感しやすいです。
